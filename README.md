# Lottttto15 — Русское Лото

Консольная реализация классической игры **Русское Лото** на Python.

## Правила

- Числа от **1 до 90** разыгрываются случайным образом из мешка (бочонки).
- Каждый игрок получает карточку с **15 числами** (3 ряда × 5 чисел).
- Вытащенный бочонок зачёркивается на карточке, если число на ней есть.
- Побеждает тот, кто **первым закроет все 15 чисел** на своей карточке.

## Структура проекта

```
Lottttto15/
├── lotto.py          # Основная логика: Card, Barrel, Player, Game
├── main.py           # Консольный интерфейс
├── requirements.txt  # Зависимости (pytest)
└── tests/
    └── test_lotto.py # Модульные тесты
```

## Запуск

```bash
python main.py
```

Игра предложит:
1. Ввести количество игроков (1–6) и их имена.
2. Выбрать режим: **автоматический** (быстрый результат) или **интерактивный** (ход за ходом).

## Тестирование

```bash
pip install -r requirements.txt
python -m pytest tests/ -v
```
