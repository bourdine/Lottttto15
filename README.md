## 📥 Как скачать APK

### Способ 1 — Страница Releases (рекомендуется)

1. Перейдите на вкладку **[Releases](../../releases)** в этом репозитории.
2. Найдите релиз **"Debug APK (Latest Build / последняя сборка)"**.
3. Раскройте раздел **Assets** и скачайте файл `app-debug.apk`.
4. Установите APK на Android-устройство (не забудьте разрешить установку из неизвестных источников в настройках).

### Способ 2 — Артефакт из GitHub Actions

1. Перейдите на вкладку **Actions** этого репозитория.
2. Нажмите на последний запуск **Build Debug APK**.
3. В нижней части страницы найдите раздел **Artifacts** и скачайте архив `app-debug-apk`.
4. Распакуйте архив — внутри будет файл `.apk`.

> **Примечание:** Для скачивания артефактов (Способ 2) требуется авторизация в GitHub.
> Для скачивания через Releases (Способ 1) авторизация не нужна.

## 📥 How to Download the APK

### Option 1 — Releases page (recommended)

1. Go to the **[Releases](../../releases)** tab of this repository.
2. Find the release named **"Debug APK (Latest Build / последняя сборка)"**.
3. Expand the **Assets** section and download `app-debug.apk`.
4. Install the APK on your Android device (enable "Install from unknown sources" in settings if prompted).

### Option 2 — GitHub Actions Artifact

1. Go to the **Actions** tab of this repository.
2. Click on the latest **Build Debug APK** run.
3. Scroll to the **Artifacts** section at the bottom and download `app-debug-apk`.
4. Unzip the archive — the `.apk` file will be inside.

> **Note:** Downloading artifacts (Option 2) requires a GitHub account.
> Downloading from Releases (Option 1) does not require login.
